This is home work for lesson 15 skillbox school "React".

Простая программа, предназначенная для отработки создания веб-приложения с применением React

Использованы следующие модули:
react
react-dom

+ webpack
+ babel-core
+ babel-loader
+ babel-preset-es2015
+ babel-preset-react

Программа позволяет добавлять и удалять комментарии. Комментарии сохраняются в локальном хранилище.
